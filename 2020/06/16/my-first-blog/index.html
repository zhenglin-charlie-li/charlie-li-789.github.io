一、 概述

  1. JavaScript是运行在客户端的脚本语言。

  2. HTML/CSS——标记语言，描述类语言，JavaScript——脚本语言，编程语言。

  3. HTML 定义了网页的内容，CSS描述了网页的布局，JavaScript控制网页的行为。

  4. 书写方法：行内式、内嵌式、引用式。

  5. 输入输出控制语句：

    alert()弹出警示框

    console.log()控制台输出

    prompt() 弹出输入框

  6. 变量：

    使用 var 声明的变量可以是字符串型，数字型，布尔型。

  2. 不用提前声明变量类型，程序会自动确定。

  3. 对同一个变量，其数据类型是可变的，例

  4. 数据类型的分类：
